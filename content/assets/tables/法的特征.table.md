
| 特征          | 内容                                                                                         |
| ----------- | ------------------------------------------------------------------------------------------ |
| 国家意志性       | 法是经过国家制定或者认可才得以形成的规范。<br>【注意】制定、认可，是国家创制法的两种方式。                                            |
| 强制性         | 法是凭借国家强制力的保证而获得普遍遵守的效力。                                                                    |
| 规范性         | ①概括性：具有能为人们提供一个行为模式、标准的属性；<br>②利益导向性：规定人们的权利和义务→分配利益→影响人们的动机和行为→影响社会关系→实现统治阶级的意志和要求、维护社会秩序 |
| 明确公开性和普遍约束性 | ①可预测；<br>②普遍使用。                                                                            |
